#Como Roda?

**Recomendo que instale dependencias dentro do ambiente virtual**

* Instale o ambiente virtual dentro da pasta **FastAPI**

- python -m venv .venv
* Após isso ative o ambiente virtual e continue abaixo
  
- pip install fastapi uvicorn
- **para rodar o código**
- uvicorn main:app --reload
